Introduction to JavaScript

JavaScript is a programming language that enables interactive web pages. It is an essential part of web development.

Basic JavaScript Syntax

                
                    // Single-line comment
                    /* Multi-line comment */

                    let variableName = "value";
                    const constantName = "constant value";

                    function functionName(parameters) {
                        // Function body
                    }

Common JavaScript Concepts

  • Variables: Used to store data values.
  • Functions: Reusable blocks of code.
  • Conditionals: if, else if, else.
  • Loops: for, while.

Example: Creating a Simple JavaScript Function

                
                    let greeting = "Hello, World!";
                    console.log(greeting);

                    function add(a, b) {
                        return a + b;
                    }

                    console.log(add(5, 3)); // Outputs: 8

Example: Manipulating the DOM

                
                    document.addEventListener('DOMContentLoaded', function() {
                        const heading = document.querySelector('h1');
                        heading.style.color = 'blue';

                        const paragraph = document.querySelector('p');
                        paragraph.textContent = 'This text has been dynamically changed by JavaScript!';
                    });

JavaScript Events

JavaScript events are actions that occur as a result of the user or the browser manipulating a page.

                
                    document.getElementById('myButton').addEventListener('click', function() {
                        alert('Button clicked!');
                    });
