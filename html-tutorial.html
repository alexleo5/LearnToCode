Introduction to HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses a series of elements to structure content on the web.

Basic Structure of an HTML Document

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>Document</title>
                    </head>
                    <body>
                        <h1>Hello, World!</h1>
                        <p>This is a paragraph.</p>
                    </body>
                    </html>

Common HTML Elements

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <a>: Anchor (for links)
  • <img>: Image
  • <ul> and <li>: Unordered list and list items

Example: Creating a Simple Web Page

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <title>My First Web Page</title>
                    </head>
                    <body>
                        <h1>Welcome to My Web Page</h1>
                        <p>This is a simple web page created using HTML.</p>
                        <img src="image.jpg" alt="Description of the image">
                        <ul>
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>
                    </body>
                    </html>

HTML Forms

HTML forms are used to collect user input. Here's an example:

                
                    <form action="/submit" method="post">
                        <label for="name">Name:</label>
                        <input type="text" id="name" name="name"><br>
                        <label for="email">Email:</label>
                        <input type="email" id="email" name="email"><br>
                        <input type="submit" value="Submit">
                    </form>
