Introduction to Go

Go (also known as Golang) is a statically typed, compiled programming language designed at Google. It is known for its simplicity, efficiency, and strong support for concurrency.

Basic Go Syntax

                
                    // Single-line comment
                    /*
                    Multi-line comment
                    */

                    package main

                    import "fmt"

                    func main() {
                        variableName := "value"
                        const CONSTANT_NAME = "constant value"

                        fmt.Println(variableName)
                    }

Common Go Concepts

  • Variables: Used to store data values.
  • Functions: Reusable blocks of code.
  • Conditionals: if, else if, else.
  • Loops: for.

Example: Creating a Simple Go Program

                
                    package main

                    import "fmt"

                    func add(a, b int) int {
                        return a + b
                    }

                    func main() {
                        greeting := "Hello, World!"
                        fmt.Println(greeting)

                        result := add(5, 3)
                        fmt.Println(result) // Outputs: 8
                    }

Example: Working with Arrays and Loops

                
                    package main

                    import "fmt"

                    func main() {
                        fruits := []string{"apple", "banana", "cherry"}

                        for _, fruit := range fruits {
                            fmt.Println(fruit)
                        }

                        // Outputs:
                        // apple
                        // banana
                        // cherry
                    }

Go Concurrency

Go has built-in support for concurrency using goroutines and channels.

                
                    package main

                    import (
                        "fmt"
                        "time"
                    )

                    func printNumbers() {
                        for i := 1; i <= 5; i++ {
                            fmt.Println(i)
                            time.Sleep(time.Millisecond * 500)
                        }
                    }

                    func printLetters() {
                        for i := 'a'; i <= 'e'; i++ {
                            fmt.Printf("%c\n", i)
                            time.Sleep(time.Millisecond * 500)
                        }
                    }

                    func main() {
                        go printNumbers()
                        go printLetters()

                        time.Sleep(time.Second * 3)
                    }
