Introduction to Ruby

Ruby is a dynamic, open-source programming language with a focus on simplicity and productivity. It has an elegant syntax that is natural to read and easy to write.

Basic Ruby Syntax

                
                    # Single-line comment
                    =begin
                    Multi-line comment
                    =end

                    variable_name = "value"
                    CONSTANT_NAME = "constant value"

                    def function_name(parameters)
                        # Function body
                    end

Common Ruby Concepts

  • Variables: Used to store data values.
  • Methods: Reusable blocks of code.
  • Conditionals: if, elsif, else.
  • Loops: for, while.

Example: Creating a Simple Ruby Program

                
                    greeting = "Hello, World!"
                    puts greeting

                    def add(a, b)
                        return a + b
                    end

                    puts add(5, 3) # Outputs: 8

Example: Working with Arrays and Loops

                
                    fruits = ["apple", "banana", "cherry"]

                    fruits.each do |fruit|
                        puts fruit
                    end

                    # Outputs:
                    # apple
                    # banana
                    # cherry

Ruby Gems

Ruby gems are libraries that can be used to extend the functionality of Ruby. For example, the sinatra gem can be used to create web applications.

                
                    require 'sinatra'

                    get '/' do
                        "Hello, World!"
                    end
